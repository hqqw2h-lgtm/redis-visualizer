📖 阅读指南

本书将 Blaze-Persistence 的核心机制拆解为 24 章，涵盖 CriteriaBuilder 查询引擎、Entity View DTO 投影、高级 SQL 特性（CTE / 窗口函数 / 集合操作）、Keyset 分页数学原理、批量 DML、Spring Data 集成、生产环境最佳实践，7 章深层原理，1 章 SQL 优化，以及 3 章 Spring Boot CRUD 实战——从零搭建电商订单管理系统，融合前 21 章全部知识。

每章以 Mermaid 架构图和流程图为主，文字仅作图的注解。遇到 JPA 原生 Criteria API 对比时，使用红绿对比块直观展示 Blaze-Persistence 的类型安全优势。

💡 推荐路径：先通读第 1-3 章建立核心认知（全景 → 搭建 → CriteriaBuilder），然后进入 Entity View 三连（第 4-6 章）。 高级 SQL 特性（第 7-9 章）和 DML（第 10-11 章）可按需跳读。Spring 集成（第 12 章）和最佳实践（第 13 章）适合实战前通读。 深层原理（第 14-20 章）揭示框架内部机制——JPQL 渲染管线、JPA Provider 钩子、代理生成、GraphQL 集成与 Quarkus 原生编译、全路径追踪与扩展体系。
优化与实战（第 21 章）覆盖 SQL 优化模式——Entity View 投影优化、N+1 检测、Keyset 分页、CTE 报表查询等。
CRUD 实战（第 22-24 章）是全书收官——用 Spring Boot + Blaze-Persistence 从零搭建完整的电商订单管理系统，涵盖项目搭建、Entity View 精确投影、动态查询、窗口函数、递归 CTE、自定义 JpqlFunction、多租户 ViewFilter、审计、Keyset 分页、批量归档、测试与监控。
