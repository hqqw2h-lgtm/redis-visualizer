25.6 致谢

本书虽以图解形式呈现，但理论根基全部来自下列贡献者的工作：

  • Christian Beikov — Blaze-Persistence 的设计师与首席维护者，本书的"灵感来源 #1"。Entity View 的优雅设计反映了对 JPA 痛点的深刻洞察。
  • Vlad Mihalcea — Hibernate 团队成员，High-Performance Java Persistence 作者。他对 JPA 性能反模式的揭示让 Blaze 的存在变得有意义。
  • Markus Winand — modern-sql.com / use-the-index-luke.com 作者。让"现代 SQL 标准"重新成为开发者的常识。
  • Erik Meijer — LINQ 设计师。"嵌入式 DSL 表达数据库查询"的开山祖师。
  • Eric Evans / Martin Fowler — DDD 与企业架构模式作者。Updatable Entity View 与 Specification 模式的概念基础。
  • Blazebit 开源社区 — 在 GitHub 上贡献了大量数据库适配、bug 修复和 Spring/Quarkus 集成改进的开发者。
  • Hibernate 团队 — Blaze 默认依赖的 JPA Provider，Christian 长期与 Hibernate 团队协作处理底层 bug。
